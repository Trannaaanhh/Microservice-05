## 📞 TROUBLESHOOTING NHANH

| Problem | Solution |
|---------|----------|
| Container crash ngày đầu | `docker compose down -v && docker compose up --build -d` |
| Database migration fail | Check logs: `docker compose logs postgres` |
| Port 8000 busy | `lsof -i :8000` then `kill -9 <PID>` |
| Services không kết nối | Wait 60s, RabbitMQ/Postgres need time |
| Mermaid render fail | Paste in https://mermaid.live |
| Load test error | Check: `docker compose ps` - tất cả phải "Up" |
